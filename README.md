## 📌 优势及注意事项

```
    使用springcloudAlibaba基础框架
    数据库使用mysql
    缓存使用redis
    rpc使用Feign
    工具类使用hutool
    网关gateway
    分布式锁redisson
    实体工具lombok
    json工具Gson
    配置中心&服务发现nacos
    消息队列rabbitMQ
    邮件服务springmail
```
