## 📌 优势及注意事项

```
    使用springcloudAlibaba进行后端开发
    前端使用vue3进行开发
    数据库使用mysql
    缓存使用redis
    代码生成器使用的人人开源
```

## 💚 适合人群
    需要有自己的博客的同学
    只需要联系作者加个账号 你就可以拥有自己的博客了
## 💚 架构
~~~
 博客系统
   web前端
    设计初衷 让身边的朋友可以使用一个博客网页用来记录自己的学习笔记或者博客
        
   web后端
      使用gateway全局入口 并且动态路由
      使用nacos注册服务与发现
      gson作为json序列化
      微服务架构
   
~~~
